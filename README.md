# Garnet — Privacy Policy

_Last updated: June 16, 2026_

Garnet is a browser extension developed by Red Summit Consulting Inc. that assists insurance underwriters and customer service representatives by reading policy-related information displayed on the page and making it available to the Garnet assistant running on the same page.

## What the extension does

The Garnet extension reads field values shown on the web page you are viewing (for example, policy and driver details displayed in your workflow) and makes that on-screen context available to the Garnet assistant on the same browser tab, so the assistant can answer questions relevant to what you are viewing.

## Data handling

- The extension makes no network requests of its own and transmits no data to any external server, including Red Summit Consulting.
- The extension does not store, save, or retain any information it reads. It reads the current page content only at the moment it is needed and passes it to the Garnet assistant on the same page.
- The extension does not collect personally identifiable information for its own purposes, does not track browsing activity, and does not use cookies.
- The extension requests no permissions beyond running on the specific website where the Garnet assistant operates.

## Data processed by the Garnet assistant

The Garnet assistant (the web application the extension supports) processes questions and on-screen context to generate answers. The handling of that data by the Garnet application is governed by the agreement between Red Summit Consulting and the organization using Garnet, and is separate from this extension.
